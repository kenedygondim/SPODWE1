Atividades da disciplina

Atividade 1 - Hospedagem no Github

Criar conta no https://github.com/, criar repositório no github da disciplina com o nome "SPODWE1" e um arquivo index.html com links para as atividades, projetos e trabalhos da disciplina, estilizar a página, publicar no githubpages e enviar na tarefa os links do repositório e do deploy.

Atividade 2 - Wireframe página pessoal

Criar uma pasta "atividades" dentro do repositório da disciplina no github e dentro desta criar uma subpasta "A2", elaborar um wireframe ou um protótipo de uma página pessoal, para isso pode se utilizar o site https://cacoo.com/templates ou https://www.figma.com/ para essa atividade, exportar a imagem do wireframe ou protótipo, colocar este arquivo gerado na pasta "A2" em seu github e adicionar o link da pasta "A2" aqui no moodl

Atividade 3 - Semântica HTML

Elaborar uma página html, para isso o aluno deverá dividir a imagem da Atividade 2 (Wireframe) em layout semântico e reproduzir a página da imagem em html, sem a necessidade de acrescentar o css. Segue dois arquivos de exemplo de como realizar essa tarefa. Não há necessidade de escrever textos completos na página, pode ter apenas textos identificando a tag na qual o texto está. O aluno deverá salvar os arquivos em uma subpasta "A3", dentro da pasta "atividades" no repositório do github da disciplina, fazer o deploy no githubpages e enviar o link do repositório e do deploy.

Atividade 10 - Máscara e validação

Assistir e executar os códigos da vídeo aula de mascara e validação em JavaScript. Após a codificação em JavaSCript, HTML e CSS o aluno deve salvar os arquivo em uma subpasta "A10", dentro da pasta "atividades" no repositório do github da disciplina, assim como, fazer deploy (publicar) a ferramenta web no GitHub Pages (ou quaisquer outro serviço) e enviar o link do repositório e da ferramenta no moddle.
